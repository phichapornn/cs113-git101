- การประกาศอาร์เรย์
int A[10];
- int คือประเภทของอาร์เรย์
- A คือ ชื่อของอาร์เรย์
- [] คือการบอกว่าการเป็นอาร์เรย์
- 10 คือ ขนาดของอาร์เรย์ A[0]-A[9]
- การเรียกใช้อาร์เรย์
ตัวแปรแต่ละตัวจะมีเบอร์กำกับ(index)อยู่
- #defineเป็นคำสั่งที่สั่งให้compilerเปลี่ยนอักษรทั้งหมดก่อนคอมไพล์
- const //ตัวแปรนี้จะไม่มีการเปลี่ยนแปลงตลอดการทำงานของโปรแกรม
ตัวอย่าง const int arraySize = 5;
- pass by value
    -ก้อปปี้ค่าของตัวแปรจากผู้เรียกไปยังฟังก์ชัน
    -สร้าตัวแปรใหม่
- pass by reference
    - ส่งตำแหน่งของตัวแปรจากผู้เรียกไปให้ฟังก์ชัน
    - ตัวแปรในฟังก์ชันเป็นตัวเดียวกันกับในผู้เรียก
- 2D array
    - อาร์เรย์สองมิติเป็นตาราง ประกาศแบบสองชั้น
